# Распив парфюмерии — одностраничник (ИП Жданов)

Статический сайт-витрина без сборки и бэкенда. Каталог из 23 ароматов на распив (3 и 5 мл),
корзина и оформление заказа через WhatsApp / Telegram / e-mail / обратный звонок.

## Структура

```
index.html              # витрина: hero, преимущества, каталог, оплата/доставка, футер
offer.html              # публичная оферта
privacy.html            # политика конфиденциальности (152-ФЗ)
assets/
  css/styles.css        # все стили (светлая минималистичная тема)
  js/config.js          # ← КОНТАКТЫ, РЕКВИЗИТЫ, КАНАЛЫ ЗАКАЗА (правится тут)
  js/products.js        # ← ТОВАРЫ И ЦЕНЫ (правится тут)
  js/app.js             # логика: рендер, корзина, оформление
  img/                  # фотографии товаров (slug-имена)
```

## Что нужно заполнить перед публикацией

### 1. Контакты и реквизиты — `assets/js/config.js`
- `shopName`, `shopTagline` — название и подзаголовок магазина;
- `phoneDisplay` — телефон как показывать; `phoneRaw` — только цифры с кодом страны
  (например `79991234567`) — используется для звонка и WhatsApp;
- `telegram` — ник без `@`;
- `email` — почта;
- `formspreeEndpoint` — (необязательно) для отправки заявок на почту через форму.
  Зарегистрируйся на https://formspree.io, создай форму, вставь её адрес
  (`https://formspree.io/f/xxxxxxxx`). Если оставить пустым — кнопка «На почту» откроет
  почтовый клиент пользователя (mailto);
- `channels` — какие способы заказа показывать (true/false);
- `legal` — ФИО ИП, ИНН, ОГРНИП, адрес (подставляются в футер, оферту и политику).

### 2. Цены — `assets/js/products.js`
У каждого товара поля `price3` (3 мл) и `price5` (5 мл). Сейчас везде `0` — это значит
«цена по запросу». Поставь числа в рублях, например:
```js
price3: 350, price5: 550,
```

### 3. Добавить / убрать товар — `assets/js/products.js`
Скопируй любой блок `{ ... }` в массиве `PRODUCTS`, поменяй поля. Картинку положи в
`assets/img/` и укажи путь в `img`.

## Запуск локально

Просто открыть `index.html` в браузере обычно работает, но для корректной работы
копирования текста в Telegram и отправки формы лучше поднять локальный сервер:

```bash
# из папки проекта
python -m http.server 8000
# затем открыть http://localhost:8000
```

## Публикация (хостинг)

Это обычные статические файлы — подойдёт любой хостинг:
- **Netlify / Vercel / GitHub Pages** — бесплатно, перетащить папку или подключить репозиторий;
- **Beget / reg.ru / Timeweb** — залить содержимое папки в каталог сайта (`public_html`) по FTP.

Никакой базы данных и сервера не требуется.

## Примечания

- Описания ароматов и ноты написаны своими словами (чужие тексты не копировались).
- Фотографии — собственные снимки продавца. При желании можно обрезать карточку с подписью
  для более «магазинного» вида.
- Объём до 5 мл включительно не подлежит обязательной маркировке «Честный знак» и сертификации.
